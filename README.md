## 使用方法
程序运行后，按照提示输入您想要转换的网址链接(需完整包含http/https)。程序支持双向转换，自动识别webVPN链接与直连地址，按回车键程序自动运行后输出访问链接，该链接地址将自动保存到系统的剪切板中，到浏览器中直接粘贴使用即可。
